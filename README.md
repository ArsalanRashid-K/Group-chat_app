# Group-chat_app
The Flutter Group Chat App is a user-friendly mobile application that enables real-time group communication. With Firebase SDK integration, users can chat in  group and exchange messages. The app supports push notifications and  works on both Android and iOS devices.

## Tools & Skills Used for this Project

* Flutter Framework: UI development for cross-platform mobile applications.
* Firebase SDK: Integration for user authentication, real-time messaging, and database management.
* Flutter Packages: Additional libraries for enhanced functionalities and UI components.
* Firebase Authentication: Secure user authentication using email .
* Firebase Realtime Database/Cloud Firestore: Real-time messaging and data storage.
